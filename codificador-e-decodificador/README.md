# Projeto CripoOracleONE

## Criptografador e Descriptografador

### Sobre o projeto

- Apenas letras minúsculas

- Não permite acentuação ou caracteres especiais
- Não permite números

### Tecnologias utilizadas

- HTML
- CSS
- Javascritpt
